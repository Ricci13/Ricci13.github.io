Sobre Mim

Sou um Analista de Dados apaixonado pela interseção entre dados, estratégia de negócios e o mercado financeiro. Atualmente focado em traduzir conjuntos de dados complexos em insights acionáveis para a Superintendência PF/PJ do Banco Mercantil.


Recentemente, co-liderei a implementação de uma cultura orientada a dados no meu setor, o que contribuiu diretamente para um crescimento de 250% da carteira. Com uma base sólida em SQL (Snowflake) e Power BI, estou expandindo ativamente meu conhecimento para Ciência de Dados, aplicando Python (Pandas, Prophet) na construção do meu primeiro modelo preditivo de Séries Temporais para gestão de liquidez.


Como estudante de Administração (PUC Minas), acredito que as melhores soluções de dados são aquelas que resolvem problemas reais, desafiam o status quo e simplificam a tomada de decisão.
